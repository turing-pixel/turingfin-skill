# TuringFin

**TuringFin** 是面向个人投资者的研究辅助产品，帮助你看懂单只股票的节奏与风险，用对话整理选股偏好，并管理自选列表。

官网：[https://www.turingfin.com](https://www.turingfin.com)

---

## 这是什么

在 TuringFin 里，你可以用自然语言或页面操作完成研究，例如：

- 「帮我分析贵州茅台，标准深度」
- 「我想做价值型选股，偏好消费行业」
- 「把平安银行加入自选」
- 「查一下 600519 现在什么价」

系统会用清晰的中文引导你下一步，完成分析、选股、自选等任务。

---

## 我能帮你做什么

### 股票分析

在股票分析页打开 **「分析配置」**，选择标的、**分析深度**（一级快速～五级全面）、**分析师团队**（市场 / 基本面 / 新闻 / 社媒，至少一人），并可开关情绪分析、风险评估与报告语言。确认后点击 **「开始分析」**，等待数分钟生成**择时研究报告**（含结论、风险提示与关键指标说明）。

- **深度越高，耗时越长**：标准分析约 4～8 分钟，全面分析可能 8～16 分钟。  
- **进行中**可查看进度条与阶段提示；离开页面后可在右上角 **「分析任务」** 查看，完成后会提醒。  
- 报告生成后可在当前页查看，或在 **历史记录** 中随时回看。  
- 常用默认值可在 **设置 → 分析设置** 中预先配置。

### 对话选股

用对话说明你的市场、行业、风格等偏好，逐步收敛条件，得到**候选股票列表**，再一键进入单票分析。

### 自选管理

把正在关注的股票放进自选列表，集中查看；不需要的标的可以随时移除。登录后列表会与账号同步。

### 搜索与行情

按股票名称或代码搜索标的；对沪深 A 股可查看最新报价与涨跌情况。

### 登录与账号

**登录后可用：** 云端股票分析、对话选股、搜索、自选与账号同步。  
**未登录时：** 自选可暂存在本机；分析、选股、搜索等云端能力不可用。

**如何登录**

1. 打开官网，进入 **登录** 页。  
2. **账号密码：** 使用 11 位手机号或用户名 + 密码（8～16 位），勾选条款后登录。  
3. **微信扫码：** 在登录页切换到微信，扫码并确认授权（二维码约 2 分钟失效，页面会自动刷新）。  
4. **新用户：** 登录页点「去注册」，填写用户名、邮箱、手机号与密码；注册成功自动登录。

**如何退出**  
工作台 → **账号** → **退出登录**，随后回到首页。

**在 Cursor 等工具里连接同一账号（可选）**  
若希望助手在对话里直接代你访问云端分析、选股等能力：登录官网后打开 **用户中心 → 连接 AI 助手**，按页面生成授权信息，将**整行**内容保存到编辑器环境变量 **`TURINGFIN_AUTHORIZATION`**（变量名以页面为准）。勿将授权内容粘贴到对话或聊天中；配置完成后重启或新建对话即可。

也可先打开 [连接助手登录入口](https://www.turingfin.com/login?intent=agent&next=%2Fapp%2Faccount%2Fconnect)（登录成功后会进入「连接 AI 助手」页）。若你不使用对话工具代访问，可忽略本节。

登录状态在同一浏览器中通常会保持；换设备或长时间未用后，需重新登录。

## 推荐怎么用

**想先缩小范围再深入一只股**  
对话选股说明偏好 → 看候选列表 → 选一只 → 生成分析报告。

**已经想好要分析哪只**  
打开分析配置 → 选标的、深度与分析师团队 → 开始分析 → 等待进度完成 → 在历史记录查看。

**平时盯几只票**  
先维护自选 → 从自选进入分析或查看行情。

## 怎么开始

### 在官网使用（推荐新用户）

1. 打开 [https://www.turingfin.com](https://www.turingfin.com)  
2. 注册或登录账号  
3. 从工作台进入「股票分析」「对话选股」「自选」等页面  

新用户也可先看站内「产品介绍」，了解能力与合规说明。

### 在 AI 助手里使用（对话完成同样的事）

TuringFin 提供可安装的 **技能包**，适用于支持 Agent Skills 的对话工具。安装后，用说话的方式即可分析、选股、管理自选；**账号与官网互通**，需要完整页面时随时打开网站即可。

**第一步：安装技能（各平台相同）**

在终端执行：

```bash
npx skills add turing-pixel/turingfin-skill -g -y
```

技能从公开仓库 [github.com/turing-pixel/turingfin-skill](https://github.com/turing-pixel/turingfin-skill) 安装；与应用主仓库是否私有无关。

安装完成后，**重启或重新打开**你使用的 AI 工具，再在设置里确认技能已启用（不同产品菜单名称可能为 Skills、技能、Agent Skills）。

**第二步：在对话里直接说需求**

新建一轮对话，例如：

- 「帮我分析一只股票」
- 「我想做价值型选股」
- 「把某某加入自选」

助手会先问好，并列出可协助的功能，你按提示继续即可。

## 常见问题

**我是新手，从哪里入手？**  
打开官网浏览「产品介绍」，再试一次对话选股或单票分析即可。

**提示需要登录怎么办？**  
打开官网登录页，用账号密码或微信扫码登录；新用户可先注册。登录成功后回到原功能继续即可。

**登录状态失效？**  
重新打开登录页登录一次；若微信扫码失败，可改用账号密码。

**微信扫码没反应？**  
等待二维码自动刷新，或刷新页面；仍不行则改用账号密码登录。

**分析报告很久没出来？**  
先对照所选深度的预计时间（标准约 4～8 分钟）；可点「分析任务」看是否仍在进行。仍无结果时，到「历史记录」查看，或重新发起一次分析。

**不知道选什么深度或分析师？**  
默认选 **三级·标准分析** 与 **市场 + 基本面** 两名分析师即可；需要事件线索可加新闻分析师。在「分析配置」弹窗中每项都有说明与预计耗时。

**对话选股没有回复？**  
检查网络后重试；也可刷新「对话选股」页面继续。

**在 DeepSeek、Kimi、通义千问 等工具里没有反应？**  
确认已执行安装命令并**完全重启**该应用；在「技能 / Skills」里查看 `turingfin` 是否已启用。部分平台需较新版本才支持 Skills。仍无效时，可先在 [官网](https://www.turingfin.com) 完成同样操作，或联系提供安装说明的服务方。
